Part 1 — Audit

Ten Criticisms to Keep

The deck now centers the model, leakage, and validation criticisms: C1–C10.

Feature & preprocessing channels
C1 · material_grouped is a paper alias
Free-text lab spellings fingerprint papers: MI(paper)=2.47 nats, grouped permutation ≈ 0.
C2 · Apparatus descriptors masquerade as physics
Chamber dimensions, facility, igniter protocol, and parse counts mostly identify rigs.
C3 · Redundant features amplify fingerprints
Dimension copies and gravity/log-gravity let column subsampling keep a barcode in every tree.
C4 · parse_core_outer misses edge patterns
Patterns like "Ø0.5/0.8 mm" reduce coverage; minor, but visible in parsing audit.
C5 · Duplicate handling is split-sensitive
Deduplication after group attachment leaves alias duplicates and random CV contamination.
Validation & training protocol
C6 · Quoted scores come from random splits
Random splits measure campaign recall, not generalisation to unseen papers.
C7 · HP search optimises the wrong objective
StratifiedKFold tuning selects memorisation capacity; group-aware search was not used for selection.
C8 · Threshold tuned on the test set
best_f1_threshold(y_test, proba_test) leaks test labels into the reported F1.
C9 · GroupKFold is unshuffled and unstratified
One deterministic partition gives unstable fold ratios with 85 papers and σ ≈ 0.07.
C10 · scale_pos_weight conflates imbalances
The 76/24 row skew is partly paper composition; one global class weight treats it as intrinsic.

Part 2 — Evals

Interventions Implemented (A–F)

6 requirements fully built and benchmarked in fable_train.py / fable_eval.py

A Paper Weights
1/N, 1/√N, effective-number (β=0.999), 1/(1+ln N) — all mean-normalised and tested.
B Combined Weights
Paper × class weights vs scale_pos_weight vs each alone. Addresses both imbalance types.
C Nested Thresholds
MCC/F1/balanced-acc/Youden J — selected on inner folds only. Vectorised over ROC operating points.
D Focal Loss (XGB)
Closed-form grad/hess (Wang et al. 2019). γ∈{1,2}. Verified: reproduces binary logistic at γ=0 (r=1.000).
E Group-Aware HP Search
StratifiedGroupKFold(5) scoring, 60 candidates. Full history CSV. Result: landscape is flat — HPs never binding.
F Domain Generalisation
Physics feature set (fingerprints removed), monotone O₂ constraint, paper-level cluster bagging (25 bags). Per-paper z-scoring rejected — destroys LOC physics.
Part 3 — Recommendations

What to Report & What to Fix Next

The ceiling is a data-content limit, not a modelling defect

✓ Recommended Evaluation Protocol
Primary: Repeated StratifiedGroupKFold(5)
≥3 shuffle seeds over canonical paper IDs. Report mean ± sd of pooled fold ROC-AUC/PR-AUC. All model selection nested inside training folds.
Secondary: LOPO with within-paper AUC distribution
Report median + IQR + count below 0.5. Separates within-paper ranking skill from between-paper calibration.
Allowed: Random splits — labelled explicitly
Report only as "within-campaign interpolation upper bound", clearly separated from the honest grouped score.
→ What Would Actually Help
Data curation over model engineering
Add papers covering novel physical regimes (lunar gravity, pure O₂, low-pressure). Current 85-paper coverage leaves large gaps in the physical parameter space.
Harmonise label definitions across papers
Separate "sustained spread" from "flash" from "self-extinguishing". Label shift is the main unfixable residual — address it in data, not model.
Deploy probabilities, not hard labels
Thresholds range 0.39–0.997 across papers. Any deployment must recalibrate per campaign or output probabilities. A global threshold is indefensible.

What the model can honestly support

This is not a failed model story. It is a measurement story: the current corpus supports within-campaign ranking, but it does not yet support a strong claim of calibrated transfer across independent papers.

Report
Grouped AUC ≈ 0.65
Use it as the primary unseen-paper claim, not the random-split upper bound.
Preserve
Within-paper ranking ≈ 0.80
The useful signal is local ordering inside campaigns, especially oxygen-driven LOC behavior.
Improve
Add independent evidence
More papers, harmonised labels, uncertainties, and thermophysical descriptors are the next bottleneck.
Bottom line: keep the grouped protocol, close the obvious fingerprint channels, and make the remaining limitation explicit. The model is informative when the claim matches what the data can identify.
